## CRITICAL RESEARCH FINDINGS

### Line Arrays vs Column Arrays Decision

**RESEARCH CONCLUSION**: **LINE ARRAYS SUPERIOR** for main coverage in reverberant space

**Academic Testing Results**:
- Church with 2-second RT: Line arrays produced **positive direct-to-reverberant ratio** at all positions
- **Best C50 results** for critical speech bands (1kHz, 2kHz, 4kHz)
- **Dramatically higher speech intelligibility** than point source in reverberant environments

**Line Array Advantages for 3.8-5.2s RT60 Environment**:
1. Precise vertical directivity control - minimizes ceiling/dome reflections
2. Minimal room excitation - avoids exciting reverberant field
3. Better low-frequency control (<400 Hz problematic frequencies)
4. Long-throw capability - 136m prayer hall requires long-throw system

**Column Array Acceptable For**:
- Column-mounted applications (slim profile, architectural sensitivity)
- Steerable dispersion addresses reverberation if beam-steering capable
- Hybrid approach: Column arrays on columns, line arrays for main coverage zones

**ORIGINAL REPORT PROPOSAL**: Renkus-Heinz IC Live, Nexo GEO column arrays
**RESEARCH VALIDATION**: ✅ Appropriate for column-mounted, ⚠️ but line arrays better for main zones

## POWER BUDGET CALCULATIONS

### Long-Throw Distance Requirements (136m Prayer Hall)

**Target**: 75 dB at 120m (rear of hall) from 16m (front speaker position)

**Inverse Square Law Calculation**:
- Distance factor: 120m / 16m = 7.5× distance
- SPL loss: ~17.5 dB (per distance doubling)
- Required SPL at source: 75 dB + 17.5 dB + 10 dB (headroom) = **102.5 dB @ 1m**
- **Conclusion**: Need high-powered line array elements (500W+ continuous per cabinet)

**Amplifier Debate**:
- **Original Proposal**: Crown XLS Series (100-200W/ch, lower cost)
- **Research Recommendation**: Crown I-Tech HD Series (1000-2000W/ch for long-throw)
- **Rationale**: 136m throw distance requires higher power than initially specified
